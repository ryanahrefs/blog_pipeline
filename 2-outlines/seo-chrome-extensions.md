## What Are SEO Chrome Extensions?
**Target**: ~150 words

- **BLUF**: SEO Chrome extensions are browser add-ons that surface SEO data and tools directly on any webpage you visit—no need to switch to separate platforms.
- They overlay metrics, analyze on-page elements, and automate repetitive checks
- Most are free or freemium with premium features
- They work passively (always-on data) or actively (click to analyze)
- **Evidence**: The Chrome Web Store has 100+ SEO-related extensions, but only a handful are worth installing

## Best Extensions for Technical SEO
**Target**: ~500 words

- **BLUF**: Technical SEO extensions help you spot issues like broken links, redirect chains, and rendering problems instantly—without running a full site crawl.

### Lighthouse
- Google's official performance and SEO auditing tool
- Scores pages on Performance, Accessibility, Best Practices, SEO
- Built into Chrome DevTools but extension makes it one-click
- **Best for**: Core Web Vitals audits, performance debugging

### Redirect Path
- Shows full redirect chain for any URL
- Color-coded status codes (301, 302, 404, 500)
- Reveals redirect loops and chains
- **Best for**: Fixing redirect issues, migration audits

### Check My Links
- Scans entire page for broken links
- Highlights valid (green) and broken (red) links
- Export results for fixing
- **Best for**: Content audits, finding link rot

### View Rendered Source
- Compare raw HTML vs JavaScript-rendered HTML
- Critical for JavaScript-heavy sites
- Shows what Googlebot actually sees
- **Best for**: Debugging indexing issues on SPAs

### Web Developer
- Swiss Army knife for inspecting page elements
- Disable CSS/JavaScript, view meta tags, outline headings
- Resize browser to test responsive design
- **Best for**: Quick technical checks, accessibility testing

## Best Extensions for Keyword Research
**Target**: ~350 words

- **BLUF**: Keyword extensions surface search volume and related terms directly in Google search results—speeding up research without leaving the SERP.

### Keyword Surfer
- Shows search volume next to Google search bar
- Displays volume for related searches
- Estimates traffic for ranking pages
- **Best for**: Quick volume checks during research

### SERP Counter / MST SERP Counter
- Adds position numbers to Google results
- See exactly where pages rank (1-100)
- Useful for manual SERP analysis
- **Best for**: Competitive analysis, rank checking

### Keywords Everywhere (Freemium)
- Volume, CPC, competition data on Google, YouTube, Amazon
- Related keywords and "People Also Search For"
- Trend charts
- **Best for**: Multi-platform keyword research (requires credits)

## Best Extensions for Link Building & Outreach
**Target**: ~300 words

- **BLUF**: Link building extensions help you evaluate link prospects and find contact information without leaving the page you're analyzing.

### Hunter
- Find email addresses associated with any domain
- Verify email deliverability
- Shows confidence score for each email
- **Best for**: Finding outreach contacts

### SimilarWeb
- Traffic estimates and traffic sources for any site
- See competitor's top referral sources
- Identify link-worthy high-traffic sites
- **Best for**: Evaluating link prospect quality

### NoFollow / Link Analyzer
- Highlights nofollow, sponsored, and UGC links
- Quick way to see which links pass equity
- **Best for**: Evaluating link opportunities, auditing link profiles

## Best Extensions for Content & On-Page SEO
**Target**: ~400 words

- **BLUF**: Content-focused extensions analyze meta tags, headings, and page structure to ensure your on-page SEO is solid before publishing.

### Detailed SEO Extension
- Comprehensive on-page report: title, meta, headings, images, links
- Schema markup viewer
- Hreflang checker
- Clean, organized interface
- **Best for**: Pre-publish content checks, competitor page analysis

### SEO META in 1 Click
- Quick view of title, description, URL, canonical
- Headers hierarchy (H1-H6)
- Image alt text audit
- Social media tags (Open Graph, Twitter Cards)
- **Best for**: Fast meta tag audits

### META SEO Inspector
- Detailed meta tag analysis
- Warns about common issues (missing tags, too long, duplicates)
- Export data for documentation
- **Best for**: Technical content audits

### Wappalyzer
- Identifies technologies used on any website
- CMS, frameworks, analytics, CDN, and more
- Useful for competitive analysis
- **Best for**: Understanding competitor tech stacks, finding CMS-specific opportunities
