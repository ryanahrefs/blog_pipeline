# Ahrefs Reference: Daily Rank Tracking

## Ahrefs Rank Tracker Product Overview

### Core Capabilities
- Track keyword rankings globally across 190+ locations
- Monitor both desktop and mobile search results
- Track position changes over time with historical data
- Compare your performance against competitors
- Monitor SERP feature presence and changes

### Key Metrics Available
1. **Visibility**: Estimated percentage of all clicks from tracked keywords going to your website
2. **Average Position**: Mean ranking position across all tracked keywords
3. **Traffic**: Estimated total organic traffic from all tracked keywords combined
4. **Share of Voice (SOV)**: Your visibility relative to competitors for tracked keywords
5. **Position Distribution**: Breakdown of rankings by position ranges (1-3, 4-10, 11-20, etc.)

### Tracking Frequency Options
- **Weekly (Default)**: Standard update frequency on all subscription plans
- **Daily (Project Boost)**: Available as an add-on per project for more frequent updates

### Keyword Calculation Formula
```
1 keyword x 1 location x 1 project = 1 tracked keyword
```
Example: Tracking 3 keywords in 2 locations = 6 tracked keywords

### Project Limits
- Maximum: 10,000 keywords per project
- Keywords exceeding plan limits become "frozen" and won't update
- Additional keywords available in batches of 500

## Best Practices for Daily Rank Tracking

### When to Use Daily Tracking
1. **Algorithm Updates**: Monitor rapid changes during Google core updates
2. **Competitive Niches**: Fast-moving industries where rankings shift frequently
3. **Active Campaigns**: During SEO sprints or content launches
4. **Time-Sensitive Businesses**: E-commerce, news, events
5. **Client Reporting**: When clients need frequent updates

### When Weekly Tracking Suffices
1. **Stable Sites**: Established domains with consistent rankings
2. **Low Competition Keywords**: Niche topics with little movement
3. **Long-Term Monitoring**: General trend analysis over months
4. **Budget Optimization**: Conserving tracked keyword allowance

### Recommended Monitoring Routine
| Frequency | Activity |
|-----------|----------|
| Daily | Check alerts, data integrity, sudden drops |
| Weekly | Analyze strategic keywords, new content indexing |
| Monthly | Consolidate trends, connect visibility to ROI |
| Quarterly | Year-over-year comparisons, strategy review |

## Ahrefs Blog Style Notes

### Writing Characteristics
- Clear, direct language with no fluff
- Data-driven claims backed by research
- Practical how-to sections with screenshots
- Use of tables and lists for scannable content
- Authoritative but conversational tone
- Includes original research or data where possible

### Content Structure Patterns
1. Hook with the problem/opportunity
2. Define key concepts clearly
3. Explain the "why" before the "how"
4. Step-by-step instructions
5. Real examples and case studies
6. Common mistakes to avoid
7. Summary/key takeaways

### Product Mention Guidelines
- Integrate naturally within educational content
- Focus on how the product solves the reader's problem
- Include specific feature descriptions
- Use screenshots to show functionality
- Don't oversell - maintain educational value
